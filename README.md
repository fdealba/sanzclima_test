# README

Running this project in localhost:

1. Clone the repo using `git clone`
``` git clone https://github.com/fdealba/sanzclima_test.git ```

2. DB setup

``` rails db:create db:migrate ```

3. Install dependencies

``` bundle && yarn ```

4. Run on port 3000

``` rails s ```


For running tests:

1. Frontend

``` yarn test ``` or ``` jest ```

2. Backend

``` rspec ```
